# Цель 5: Гендерное равенство

<p align="center"><img src="https://www.un.org/sustainabledevelopment/wp-content/uploads/2019/08/E-Goal-05-1024x1024.png" width="200" height="200"/></p>

ОАЭ добились значительного прогресса в расширении прав и возможностей женщин и содействии гендерному равенству в различных секторах.

## Ключевые инициативы

-** Женский день Эмиратов **: празднует достижения женщин -эмиратов и способствует гендерному равенству.
-** Политики равных возможностей **: обеспечивает равную оплату и возможности для женщин в рабочей силе.
