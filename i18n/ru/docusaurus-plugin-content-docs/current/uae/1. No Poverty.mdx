# Цель 1: Без бедности

<p align="center"><img src="https://www.un.org/sustainabledevelopment/wp-content/uploads/2019/08/E-Goal-01-1024x1024.png" width="200" height="200"/></p>

ОАЭ привержены искоренению бедности и повышению качества жизни для всех его жителей.Посредством различных программ социального обеспечения и экономических инициатив ОАЭ стремится сократить бедность и обеспечить экономическую безопасность.

## Ключевые инициативы

-** Программы социального обеспечения **: ОАЭ предлагает различные программы финансовой помощи для семей с низким доходом и людей с особыми потребностями.
-** Программы создания рабочих мест **: ОАЭ поддерживает инициативы по созданию рабочих мест для предоставления стабильных возможностей дохода для всех.

Исследуйте, как ОАЭ продолжает способствовать экономическому включению в устойчивое будущее.
