# Цель 17: Партнерство для целей

<p align="center"><img src="https://www.un.org/sustainabledevelopment/wp-content/uploads/2019/08/E-Goal-17-1024x1024.png" width="200" height="200"/></p>

ОАЭ посвящено глобальным партнерским отношениям, которые стимулируют устойчивое развитие и достижение ЦУР.

## Ключевые инициативы

-** Международная помощь в развитии **: финансовая и техническая поддержка устойчивых проектов в развивающихся странах.
-** Партнерство по SDG **: Сотрудничество с международными организациями для содействия устойчивой практике.
