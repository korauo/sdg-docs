# Цель 13: Климатическое действие

<p align="center"><img src="https://www.un.org/sustainabledevelopment/wp-content/uploads/2019/08/E-Goal-13-1024x1024.png" width="200" height="200"/></p>

ОАЭ стремится к решению проблемы изменения климата посредством проактивной политики и экологических программ.

## Ключевые инициативы

-** Национальный план изменения климата **: Дорожная карта для сокращения выбросов и перехода к экономике с низким уровнем углерода.
-** Международные партнерства **: Сотрудничество с другими странами для решения проблем изменения климата.

Исследуйте, как ОАЭ предпринимает действия для борьбы с изменением климата.
