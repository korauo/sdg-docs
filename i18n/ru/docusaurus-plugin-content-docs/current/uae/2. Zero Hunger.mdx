# Цель 2: нулевой голод

<p align="center"><img src="https://www.un.org/sustainabledevelopment/wp-content/uploads/2019/08/E-Goal-02-1024x1024.png" width="200" height="200"/></p>

ОАЭ приняли несколько стратегий для улучшения продовольственной безопасности и содействия устойчивому сельскому хозяйству, несмотря на его засушливой климат.

## Ключевые инициативы

-** Национальная стратегия продовольственной безопасности **: фокусируется на улучшении цепочек поставок продовольствия и устойчивой сельскохозяйственной практике.
-** Инициативы по продовольственной безопасности **: Включает программы по сокращению пищевых отходов и увеличению производства продуктов питания.

Эти усилия согласуются с глобальными усилиями по искоренению голода и обеспечению доступности продуктов питания.
