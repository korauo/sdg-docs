# 17. Партнерство для целей
<p align="center"><img src="https://www.un.org/sustainabledevelopment/wp-content/uploads/2019/08/E-Goal-17-1024x1024.png" width="200" height="200"/></p>

##**Цель**

**СДГ 17 стремится укрепить средства реализации и оживить глобальное партнерство для устойчивого развития.**

 ## Цели: 
    
 ### 17.1. Усилить мобилизацию внутренних ресурсов для улучшения внутренних потенциал для сбора налогов и других доходов. 

 ### 17.2. Реализовать и реализовать глобальную стратегию развития и оперативно использовать эффективные партнерские отношения. 

 ### 17.3. Улучшить международную поддержку для реализации эффективного и целевого наращивания потенциала в развивающихся странах.
