# 安全理事会(SC)

<p align="center"><img src="https://www.mun.bme.hu/wp-content/uploads/2018/01/Security-Council.png" width="200" height="200"/></p>

##**客观的**

**安全理事会是联合国六个主要机构之一。 它的主要责任是维持国际和平与安全。**
## 目标：
### SC.1确定对和平或侵略行为的威胁的存在，并为其解决方案推荐策略。 

### SC.2建立维持和平行动和国际制裁，以恢复国际和平与安全。

### SC.3建议入学新会员国，并提议任命秘书长。
