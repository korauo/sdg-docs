# 目标11：可持续城市和社区

<p align="center"><img src="https://www.un.org/sustainabledevelopment/wp-content/uploads/2019/08/E-Goal-11-1024x1024.png" width="200" height="200"/></p>

阿联酋正在开发具有绿色空间，高效运输和环保建筑的可持续城市地区。

## 关键举措

-  **可持续的城市规划**：减少污染和促进绿色建筑实践的倡议。
-  **公共交通系统**：扩大地铁和公共交通选择以减少碳排放。
