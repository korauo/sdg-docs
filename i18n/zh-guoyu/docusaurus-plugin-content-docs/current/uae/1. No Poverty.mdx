# 目标1：没有贫困

<p align="center"><img src="https://www.un.org/sustainabledevelopment/wp-content/uploads/2019/08/E-Goal-01-1024x1024.png" width="200" height="200"/></p>

阿联酋致力于消除贫困，并提高所有居民的生活质量。通过各种社会福利计划和经济计划，阿联酋旨在减少贫困并提供经济安全。

## 关键计划

-  **社会福利计划**：阿联酋为低收入家庭和有特殊需要的人提供了各种经济援助计划。
-  **创建工作计划**：阿联酋支持创建工作的计划，为所有人提供稳定的收入机会。

探索阿联酋如何继续为可持续的未来促进经济包容。
