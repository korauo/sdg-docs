# 2.零饥饿

<p align="center"><img src="https://www.un.org/sustainabledevelopment/wp-content/uploads/2019/08/E-Goal-02-1024x1024.png" width="200" height="200"/></p>

##**客观的**

**SDG 2专注于实现粮食安全，改善营养和促进可持续农业。 它认识到结束饥饿并确保所有人获得营养和足够食物的至关重要。 这个目标强调了对可持续农业实践，对农村基础设施的投资以及整个食品价值链中食物浪费的需求**

## 目标

### 2.1。 结束饥饿并确保为所有人提供安全，营养和足够的食物

### 2.2。 结束所有形式的营养不良，包括发育迟缓和浪费儿童
