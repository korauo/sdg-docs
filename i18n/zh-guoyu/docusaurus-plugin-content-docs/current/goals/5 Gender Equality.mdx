# 5.性别平等

<p align="center"><img src="https://www.un.org/sustainabledevelopment/wp-content/uploads/2019/08/E-Goal-05-1024x1024.png" width="200" height="200"/></p>

##**客观的**
**SDG 5旨在实现性别平等并增强所有妇女和女孩的能力。 它认识到，性别平等不仅是人类的基本权利，而且是一个和平，繁荣和可持续世界的必要基础。 该目标解决了基于性别的歧视和暴力，同时促进了平等获得教育，经济机会和领导职务的机会。**

 ## 目标： 
    
 ### 5.1。 结束各种形式的歧视和对妇女和女孩的暴力行为。 

 ### 5.2。 消除有害的做法，例如童婚和女性生殖器肢解。 

 ### 5.3。 确保妇女充分有效地参与政治，经济和公共生活。
