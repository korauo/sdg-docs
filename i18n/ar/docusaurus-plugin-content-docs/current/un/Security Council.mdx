# مجلس الأمن (SC)

<p align="center"><img src="https://www.mun.bme.hu/wp-content/uploads/2018/01/Security-Council.png" width="200" height="200"/></p>

## **موضوعي**

**مجلس الأمن هو أحد الأعضاء الرئيسية الستة للأمم المتحدة. مسؤوليتها الأساسية هي الحفاظ على السلام والأمن الدوليين.**

 ## الأهداف: 
    
 ### SC.1 حدد وجود تهديد للسلام أو فعل العدوان والتوصية باستراتيجيات لحلها. 

 ### SC.2 إنشاء عمليات حفظ السلام والعقوبات الدولية لاستعادة السلام والأمن الدوليين. 

### SC.3 أوصي بقبول الدول الأعضاء الجديدة واقتراح تعيين الأمين العام.
